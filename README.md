## 📝 Description & Objectives
This repository serves as a centralized portfolio to showcase the technical skills and professional certifications acquired through the partnership between **Holberton School France** and **IBM**. 

The primary objectives are:
- To provide a structured record of academic progress.
- To demonstrate expertise in various domains such as Cloud, Agile, SQL, and Cybersecurity.
- To facilitate the sharing of credentials with recruiters and peers.

## 🛠 Technologies Used
- **Document Formats:** PDF (Portable Document Format)
- **Image Formats:** PNG (Portable Network Graphics)
- **Documentation:** Markdown

## ⚙️ Prerequisites
To view the certificates, you will need:
- **OS:** Linux, macOS, or Windows.
- **PDF Reader:** [Adobe Acrobat Reader](https://get.adobe.com/reader/), [Foxit Reader](https://www.foxit.com/pdf-reader/), or a modern web browser (Chrome, Firefox, Edge).
- **Image Viewer:** Any native OS image viewer.

## 🚀 How to Install
Clone this repository to your local machine:
```bash
git clone https://github.com/holbertonschool/holbertonschool-france-certificates-ibm.git
cd holbertonschool-france-certificates-ibm
```

## 📖 How to Use
Navigate through the trimester folders to access the specific certificates:
```bash
# Example: Viewing a certificate from Trimester 1
xdg-open certificates-trimester-1/certificate-agile.pdf
```
